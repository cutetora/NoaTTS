# 音声サンプルのクレジット・出典

## ボイスクローン サンプル
`clone_before_tsukuyomi.wav`（合成前の人間の声）および `clone_after_*.wav`
（クローン生成音声）には、フリー素材キャラクター「つくよみちゃん」の
サンプルボイスを参照音声として使用しています。
NoaTTS という音声合成ソフトを紹介する目的での掲載です。

> 使用素材：つくよみちゃんのサンプルボイス
> https://tyc.rei-yumesaki.net/material/voice/sample-voice/
> （CV.夢前黎 / つくよみちゃんコーパス準拠）

つくよみちゃんのサンプルボイスは、音声合成・ボイスクローンの素材として
利用でき、ソフト紹介の文脈での掲載も規約で認められています
（素材としての再配布・転売は禁止）。

## その他のサンプル
- `01_standard_noa.mp3` … ボイス「ノア」(自作クローンボイス)
- `01_standard_male_nathan.wav` / `nathan_line_*.wav` … ボイスデザインで作成した
  オリジナル男性ボイス「ネイサン」(実在の人物・キャラクターとは無関係)
- `02〜05_emotion_*.mp3` … ノアに感情絵文字を付与した感情表現サンプル
- `male_*.wav` … ボイスデザインのプロンプト比較用(没候補含む)
